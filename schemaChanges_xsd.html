Element Addition
Namespace http://www.legislation.gov.uk/namespaces/legislation

Element Repeal
Namespace http://www.legislation.gov.uk/namespaces/legislation
Annotations
Indicates the repeal of a part of the text. This is usually displayed as dots (three for inline text and a line of dots for a repealed block). If the SubstitutionRef attribute is present, this indicates a substitution; the Addition referenced by the SubstitutionRef attribute holds the replacement text.
