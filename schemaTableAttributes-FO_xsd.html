Simple Type TableColorType
Namespace http://www.w3.org/1999/XSL/Format
Annotations

FO colour attribute values should be defined using CSS basic colour values or RGB notation

  • aqua - #00ffff
  • black - #000000
  • blue - #0000ff
  • fuchsia - #ff00ff
  • gray - #808080
  • green - #008000
  • lime - #00ff00
  • maroon - #800000
  • navy - #000080
  • olive - #808000
  • purple - #800080
  • red - #ff0000
  • silver - #c0c0c0
  • teal - #008080
  • white - #ffffff
  • yellow - #ffff00
  • hexadecimal value e.g. #4b0082 is indigo
